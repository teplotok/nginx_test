Как это сделано?

При нажатии кнопки "Отправить" текст из поля ввода берется функцией hello() и формируется в строку вида "Привет, name!" значение которой присваивается пустому параграфу, который после этого становиться видимым.

Аналогично первому примеру, по нажатию кнопки значения передаются в функцию, где происходит вычисление по формуле и результат записывается в пустой параграф.

При получении строк, происходит сравнивание по длине(.length)-параметру, который есть у каждой строки.

При получении массива,его первый элемент заносится как максимальный и минимальный одновременно, обходя массив с помощью for (var i = arr.length - 1; i >= 0; i--) мы сравниваем каждый элемент с минимумом и максимумом, заменяя в случае нового максимума/минимума.

Данный таймер построен на использовании функции setInterval(() => {sec++;updateTimer()}, 1000); которая вызывается при нажатии кнопки Старт.Первым параметром setInterval является другая функция(конкретно здесь, увеличение колличества пройденных секунд каждую секунду и обновление таймера на странице) а вторым параметров является время,через которой функция вызывается снова(так как время указывается в миллисекундах, здесь указано значеник в 1000 мс что равно одной секунде).setInterval будет бесконечно вызывать функцию через указанный промежуток, пока его не остановят извне.Кнопка Стоп немедленно останавливает дальнейший вызов благодаря функции clearInterval(timerId); где timerId это та setInterval, которая была вначале запущена. Кнопка сброс работает аналогичным образом, только в этом случае данные и прошедшем времени стираются и таймер обновляется с нулевым временем.

Тест построен на вопросах, каждый из которых представляет из себя div блок с именнованым input внутри.Внутри каждого блока есть несколько input, которые связаны между собой тегом name и radio. Первый тег обьединяет несколько обьектов в группу, второй указывает, что выбрать можно лишь один из них. Проверка теста основана на том, что на момент выполнения сценария внутри уже есть массив, в котором последовательно хранятся номера правильных ответов. Один за другим, вызываются input и если он был выбран(то есть его атрибут checked равен true)то он помечается как правильно выбранный(подсвечивается зеленым) и увеличивается счетчик правильных ответов. Иначе же правильный input подсвечивается красным и счетчик не увеличивается.

Баннер, который всплывает при нажатии кнопки, изначально невидим(display:none). После того как кнопка была нажата display меняется на flex, и с помощью свойства position:fixed фиксируется, благодаря чему он не сьезжает при прокрутке страницы. Дата формируется из обьекта new Date() но, так как он содержит много лишней информации, здесь были использованы методы getDate() getMonth() getFullYear() c помощью который и была составлена нужная дата. Она была присоединена к строке вместе с именем, которое берется из параграфа из примера 1.

JavaScript - это язык программирования, активно использующийся в современной WEB-разработке.Основной его задачей является добавление интерактивности веб-страницам.Он используется как во frontend, так и в backend разработке. С его помощью на сайт можно добавить анимации,автозаполнение, всплывающие формы-в общем контент, который меняется "на лету". Однако, JavaScript сам по себе не может создать сайт или стилизовать что-либо - это работа для HTML и CSS.
